Переменные

Переменная – это «именованное хранилище» для данных. Мы можем использовать переменные для хранения товаров, посетителей и других данных. Для создания переменной в JavaScript используйте ключевое слово let

После создания можно поместить в нее данные, с помощью оператора присваивания - "=".

Переменные можно обьявлять в 1 строку, но для лучшей читемости следует объявляйте каждую переменную на новой строке.

Пример

1 let name="Alex";
2 let age=20;

В старых скриптах вы также можете найти другое ключевое слово: var вместо let:

1 var firstString="Hello World!";

Ключевое слово var – почти то же самое, что и let. Оно объявляет переменную, но немного по-другому, «устаревшим» способом. Есть тонкие различия между let и var, но они пока не имеют для нас значения.

Константы

Чтобы объявить константную, то есть, неизменяемую переменную, используйте const вместо let:

1 const pi=3,141;

Переменные, объявленные с помощью const, называются «константами». Их нельзя изменить. Попытка сделать это приведёт к ошибке.

Типы данных

Число

Числовой тип данных (number) представляет как целочисленные значения, так и числа с плавающей точкой.
Существует множество операций для чисел, например, умножение *, деление /, сложение +, вычитание - и так далее.

Пример

Запустить
1 let x=1;
2 let y=8;
3 alert(x+y);

Строка

Строка (string) в JavaScript должна быть заключена в кавычки.

1 let str1="Двойные кавычки.";

3 let str3=`С помощью обратных кавычек можно встраивать переменные. ${str2}`;

Так же в обратные кавычки можно встраивать выражения

Запустить

Логический тип

Логический(булевый) тип (boolean) может принимать только два значения: true (истина) и false (ложь).

Такой тип, как правило, используется для хранения значений да/нет: true значит «да, правильно», а false значит «нет, не правильно».

1 let loginChecked=true;
2 loginChecked=false;

Булевые значения также могут быть результатом сравнений:

Запустить
1 alert(2<3);

alert

С этой функцией мы уже знакомы. Она показывает сообщение и ждёт, пока пользователь нажмёт кнопку «ОК».

Это небольшое окно с сообщением называется модальным окном. Понятие модальное означает, что пользователь не может взаимодействовать с интерфейсом остальной части страницы, нажимать на другие кнопки и т.д. до тех пор, пока взаимодействует с окном. В данном случае – пока не будет нажата кнопка «OK».

prompt

Функция prompt отображает диалоговое окно с необязательным запросом на ввод текста.

Запустить
1 let name= prompt("Введите имя", "имя");
2 alert("Ваше имя");

confirm

Функция confirm отображает диалоговое окно, которое содержит две кнопки (OK и Cancel), а так же опциональное (необязательное) текстовое сообщение.

Результат – true, если нажата кнопка OK. В других случаях – false.

Запустить
1 let quest= confirm("Вы изучаете js?");
2 alert(quest);

Операторы сравнения

Многие операторы сравнения известны нам из математики.

  • Больше a>b и меньше a<b;
  • Больше или равно a>=b и меньше или равно a<b=;
  • Равно a==b или a===b, при сравнении с помощью == сравниваются только значения, а с помощью === еще и типы
  • Не равно a!=b или a!==b(тот же принцип как с == и ===)
Запустить
1 alert(5>2);
2 alert(5===5);
3 alert(5!==5);

Оператор if

Оператор if(...) вычисляет условие в скобках и, если результат true, то выполняет блок кода.

В нутри оператора if можно использовать все операторы сравнения

Запустить
1 let example= prompt("2+2= ");
2 if(example==4){alert("Это верно!")};

Инструкция if (…) вычисляет выражение в скобках и преобразует результат к логическому типу.
Число 0, пустая строка "", null, undefined и NaN становятся false. Из-за этого их называют «ложными» («falsy») значениями.
Остальные значения становятся true, поэтому их называют «правдивыми» («truthy»).

1 if(1){...};
2 if(0){...};

Первый код будет выполняться всегда, а второй никогда.

Блок else

Оператор if может содержать необязательный блок «else» («иначе»). Он выполняется, когда условие ложно.

Запустить
1 let example= prompt("2+2= ");
2 if(example==4){alert("Это верно!")};
3 else{alert("Вы ошиблись.")};

Так же с помощью оператора «if» и блока «else», можно построить цепочку из условий.
Запустить
1 let example= prompt("2+2= ");
2 if(example==4){alert("Это верно!");}
3 else{
4   if(example<4) {
5    alert("Слишком мало.");
6  }else {
7    if(example>4) {
8     alert("Слишком много.");
9   };

Логические операторы

JavaScript поддерживает три логических оператора: || (ИЛИ), && (И) и ! (НЕ). Благодаря логическим операторам можно объединять два выражения в одно более сложное.

Основными логическими или булевыми операциями, названными в честь одного из математиков — Джорджа Буля (1815-1864), являются:

  • ИЛИ || – логическое сложение (дизъюнкция) – OR;
  • И && – логическое умножение (конъюнкция) – AND;
  • НЕ ! – логическое отрицание (инверсия) – NOT.

Логические операторы возращают true или false.

Логическое и ( &&, and )

В программировании И возращает true, если оба аргумента истенны, в противном случае - false:
Запустить
1 alert(true && true); //true
2 alert(true && false); //false
3 alert(false && true); //false
4 alert(false && false); //false

Логическое или ( ||, or )

В программировании ИЛИ возращает true, если хотябы 1 равен true, в противном случае - false:
Запустить
1 alert(true || true); //true
2 alert(true || false); //true
3 alert(false || true); //true
4 alert(false ||false); //false

Логическое не ( !, not )

Данный оператор принимает один аргумент, после приводит его к логическому типу true/false.
Затем возращает противоположное значение.
Запустить
1 alert(!true); //false
2 alert(!false); //true
3 alert(!1); //false
4 alert(!0); //true

Примеры с if
Запустить
1 let example= prompt("Введите число от 0 до 20");
2 if(example<15 && example>5) {
3  alert("Ваше число между 5 и 15 ");
4 }
5 if(!(example<15 && example>5)){
6  alert("Ваше число не находится между 5 и 15 ");
7 }
8 if(example==15 || example==5){
9  alert("Вы ввели 5 или 15 ");
10 }

Циклы


Циклы - простой способ сделать какое-то действие несколько раз. При этом однократное выполнения кода в цикле называется итерацией.

На языке JavaScript существуют различные виды циклов, которые по сути дела выполняют одно и тоже. Просто одни задачи решаются более просто с помощью одних циклов, а другие – с помощью других.

В JavaScript существуют различные виды циклов, но все они, по сути, делают одно и тоже. Просто с помощью одних циклов более просто решаются одни задачи, с помощью других – иные.

Цикл for

Данный цикл в основном используется когда известно точное количество повторений. Этот цикл ещё называют циклом со счётчиком.

Синтаксис цикла «for»:
1 for (инициализация; условие; финальное выражение)
2  тело цикла
3 }

Основные части конструкции цикла «for»:

  • инициализация - это выражение, которое выполняется один раз перед выполнением цикла; обычно используется для инициализации счётчика;
  • условие - это выражение, истинность которого проверяется перед каждой итерацией; если выражение вычисляется как истина, то выполняется итерация; в противном случае цикл «for» завершает работу;
  • финальное выражение - это выражение, которое выполняется в конце каждой итерации; обычно используется для изменения счетчика;
  • тело цикла - инструкции, выполнение которых нужно повторять.
1 for (let i = 1; i <= 3; i++)
2  alert(i);
3 }

В этом примере:

  • инициализация: var i = 1 (объявление переменной i и присвоение ей значения 1);
  • условие выполнения цикла: i <= 8 (пока значение переменной i меньше или равно 8);
  • финальное выражение, которое нужно выполнять в конце каждой итерации: i++ (увеличение значение переменной i на 1);
  • инструкция, которую нужно выполнять: console.log(i) (выведение значения счётчика в консоль).

Цикл while

Данный цикл предназначен для многократного выполнения одних и тех же инструкций до тех пор, пока истинно некоторое условие. Цикл «while» в основном используется, когда количество повторений заранее не известно.
1 while (условие)
2  тело цикла
3 }

Истинность условия проверяется перед каждым выполнением. Если перед первой итерацией условие ложно, то цикл не выполнится ни разу.
1 let a=1;
2 while (a<=3)
3  alert(a);
4  a++;
5 }

Цикл do...while

Цикл «do...while», также как и цикл «while», выполняет одни и те же инструкции до тех пор, пока указанное условие истинно. Но в отличие от «while» в «do...while» условие проверяется после выполнения инструкций. Поэтому цикл «do...while» в любом случае выполнится не меньше одного раза, даже если условие изначально ложно.
1 do {
2  тело цикла
3 }while(условие)

JavaScript

JavaScript является преимущественно клиентским языком, обычно использующимся для работы на стороне клиента. С его помощью можно разрабатывать приложения с самым разнообразным функционалом. Это может быть: аналог механических часов, различная анимация, графические эффекты и многое другое. Ничего не мешает создать практически все веб-приложение на JS.


Для создания кода на JS нужно всего лишь текстовый редактор и браузер, в котором можно тестировать результат.
Несколько хороших редакторов:Atom, Visual Studio Code, Sublime Text
